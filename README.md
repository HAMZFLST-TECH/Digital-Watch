# ⌚ Digital-Watch

A sleek, minimalist standby-mode digital watch interface built with **Pure Vanilla Web Tech**.

## ✨ Features
- **Standby Aesthetic:** Bold, high-contrast UI for desk-clock use.
- **Live Updates:** Real-time clock engine powered by JavaScript.
- **Pure Stack:** 0% Frameworks. 100% HTML, CSS, & JS.
- **Responsive:** Fully optimized for mobile and desktop screens.

## 🚀 Quick Start
```bash
git clone https://github.com/HAMZFLST-TECH/Digital-Watch.git
cd Digital-Watch
open index.html
```
